
CONTENTS OF THIS FOLDER:

This folder contains all files for Project6. There are two folders, one is for code for with symbols and one for code without symbols. The executable is in src/ directory.

HOW TO RUN THE CODE:
To run the code, direct to the src/ directory, input in terminal the command:
python3 project6.py <filename>.asm
The output .hack will be written to the same directory after execution.


WHAT WORKS IN MY CODE:
I have tested my code with CPU Emulator (for running the program) and Assembler provided (for comparing with the in-built translator).
